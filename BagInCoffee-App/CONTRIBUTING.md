# Contributing to BagInCoffee App

기여에 관심을 가져주셔서 감사합니다! 🎉

## 개발 환경

```bash
# 1. Flutter 설치 확인
flutter doctor

# 2. 포크 후 클론
git clone https://github.com/<your-username>/BagInCoffee_Flutter.git
cd BagInCoffee_Flutter

# 3. 의존성 설치
flutter pub get

# 4. 실행
flutter run
```

## 커밋 컨벤션

```
feat: 새로운 기능
fix: 버그 수정
docs: 문서 수정
style: UI 스타일 변경
refactor: 리팩토링
test: 테스트 추가/수정
chore: 빌드/설정 변경
```

## Pull Request

1. 이슈 생성
2. 브랜치 생성 (`feat/amazing-feature`)
3. 분석 실행 (`flutter analyze`)
4. PR 생성

## 코드 스타일

- Dart 공식 스타일 가이드
- Riverpod 상태 관리
- 위젯 분리 원칙
